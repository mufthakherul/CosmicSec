## 🎨 Color Scheme

### Primary Colors
- **Cyan/Electric Blue** (`#00D9FF`): Technology, security, trust
  - Used for: Primary branding, logos, highlights
- **Deep Purple** (`#7B2CBF`): Innovation, AI, premium
  - Used for: AI features, secondary branding
- **Dark Space Blue** (`#0A0E27`): Professional, cosmic theme
  - Used for: Backgrounds, containers

### Secondary Colors
- **Emerald Green** (`#10B981`): Success, protection, active
- **Amber Orange** (`#F59E0B`): Warnings, attention
- **Red Alert** (`#EF4444`): Critical, threats, errors

### Neutral Colors
- **Pure White** (`#FFFFFF`): Text on dark backgrounds
- **Light Gray** (`#E5E7EB`): Secondary text
- **Dark Gray** (`#1F2937`): Text on light backgrounds

## ✍️ Typography & Voice

### Tone of Voice
- **Professional**: Enterprise-grade, authoritative
- **Modern**: Tech-forward, innovative
- **Accessible**: Clear, jargon-free when possible
- **Confident**: Security-focused, trustworthy

### Writing Style
- Use active voice
- Be concise and clear
- Technical accuracy is paramount
- Balance professionalism with approachability

### Example Messaging

**Good** ✅:
- "CosmicSec empowers security teams with Helix AI intelligence"
- "GuardAxisSphere provides comprehensive threat visibility"
- "Helix AI analyzes vulnerabilities in real-time"

**Avoid** ❌:
- "World's best hacking tool" (unprofessional)
- "Ultimate solution" (marketing hyperbole)
- "Military-grade" (overused, vague)

## 🏷️ Brand Usage Guidelines

### When to Use Each Brand

**CosmicSec**:
- Package/module names: `cosmicsec`, `from cosmicsec import`
- GitHub repository references
- Official documentation titles
- Package managers (pip, npm, etc.)
- Technical specifications
- API documentation

**GuardAxisSphere**:
- User-facing web interface
- Marketing materials
- Product descriptions
- User dashboard headers
- Customer communications
- Enterprise sales materials

**Helix**:
- AI feature descriptions
- Chatbot interface
- Intelligent automation features
- AI-powered analysis tools
- Machine learning capabilities
- Natural language interactions

### Combined Usage Examples

**Correct** ✅:
- "CosmicSec's GuardAxisSphere platform, powered by Helix AI"
- "Install CosmicSec to access GuardAxisSphere and Helix AI features"
- "GuardAxisSphere (part of CosmicSec) uses Helix AI for threat detection"

**Avoid** ❌:
- Using brand names interchangeably
- Mixing branding within the same context
- Inconsistent capitalization

### Package Names
- **Python**: `cosmicsec` (lowercase, no spaces)

- **Docker**: `cosmicsec/platform:latest`

```

├── cosmicsec-core          # Main platform

```

## 🔤 Capitalization Rules

### CosmicSec
- **Code**: `cosmicsec` (all lowercase)
- **Prose**: `CosmicSec` (CamelCase)
- **Never**: `Cosmic Sec`, `cosmic sec`, `COSMICSEC`

### GuardAxisSphere
- **Code**: `guardaxissphere` or `guard_axis_sphere`
- **Prose**: `GuardAxisSphere` (CamelCase, no spaces)
- **Never**: `Guard Axis Sphere`, `guardaxis`, `GAS`

### Helix
- **Code**: `helix` (all lowercase)
- **Prose**: `Helix` or `Helix AI` (proper case)
- **Never**: `HELIX`, `helix ai`
