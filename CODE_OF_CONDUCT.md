# Code of Conduct

## Our Pledge

CosmicSec exists to advance **ethical cybersecurity** — protecting systems, educating defenders, and empowering authorized security professionals. As contributors and maintainers, we pledge to make participation in this project a respectful, harassment-free experience for everyone, regardless of background, experience level, or identity.

We also pledge to uphold the **ethical use standards** in the [LICENSE](LICENSE): this community does not support, enable, or tolerate offensive, harmful, or unauthorized security activity.

---

## Our Standards

### Expected Behaviour

- Be respectful and constructive in all communications
- Assume good faith — ask questions before drawing conclusions
- Give and receive feedback graciously
- Focus discussions on ideas, code, and security concepts — not on individuals
- Respect the ethical boundaries of this project: only discuss techniques in the context of **authorized, ethical security work**
- Protect others' privacy — do not share personal information without consent
- Report security concerns through proper channels (see [SECURITY.md](SECURITY.md))

### Unacceptable Behaviour

- Harassment, intimidation, or personal attacks of any kind
- Sharing or soliciting malware, unauthorized exploit code, or tools designed for unethical offensive use
- Discussing, glorifying, or assisting unauthorized access to third-party systems
- Doxing, sharing private contact information, or targeting individuals
- Discriminatory language or jokes related to race, gender, religion, nationality, disability, or any other protected characteristic
- Spamming, trolling, or derailing discussions
- Any conduct that violates the [LICENSE](LICENSE) ethical use restrictions

---

## Scope

This Code of Conduct applies to all project spaces: GitHub issues, pull requests, code reviews, discussions, and any other project-affiliated communication channel.
