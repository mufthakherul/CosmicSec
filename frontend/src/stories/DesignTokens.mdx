Core visual primitives used across dashboard surfaces and shared components.

## Typography

- `text-sm` + `font-medium` for most control labels.
- `text-xs` + `tracking-wide` for metadata and secondary labels.
- `font-semibold` for KPI values and card headings.

## Radius, Shadows, and Motion

- Radius: `rounded-md` for controls, `rounded-xl` for cards.
- Elevation: `shadow-sm` on cards, stronger elevation only for dialogs/menus.
- Motion respects reduced-motion mode and uses short transitions (`duration-150` to `duration-200`).
