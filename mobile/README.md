# CosmicSec Mobile Companion (React Native)

Optional companion client for quick SOC visibility and scan status on mobile.

## Stack

- Expo + React Native + TypeScript

## Quick start

```bash
cd mobile
npm install
npm run start
```

## Scope

- Dashboard summary cards
- Recent scans feed
- Timeline quick triage view
- API base URL configuration for self-hosted deployments

This app is optional and does not block web/PWA usage.
